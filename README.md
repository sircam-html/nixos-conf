## NixOS Configuration Overview

💡 About this set up

- Minimalist configuration of my NixOS 25.11 environment featuring KDE Plasma 6 as main desktop and SDDM with automatic login. Powered by proprietary NVIDIA drivers with full hardware acceleration including 32-bit gaming support. Optimized for desktop use with TLP and thermald for power/thermal management, sleep functions disabled. Home Manager for pure declarative apps installation with unfree packages enabled, daily Nix garbage collection, Flakes activated, and services.flatpak.enable completely removed for 100% Nix purity.
